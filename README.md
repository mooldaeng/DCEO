# 📈 DCEO - 스마트 주식현황판 & 로또픽 (LottoPick)

본 프로젝트는 주식 초보자부터 숙련자까지 한눈에 시장 흐름을 파악할 수 있는 **'주식현황판'**과 재미있는 통계 기반 로또 번호 생성기인 **'로또픽'**을 포함한 종합 대시보드 웹 애플리케이션입니다.

---

## 🚀 주식현황판 (StockBoard) 가이드

주식 공부를 이제 막 시작하셨나요? 복잡한 HTS/MTS 대신, 핵심 정보만 깔끔하게 모았습니다.

### 1. 주요 기능 설명
- **자산 카테고리 탭**: '국내주식', '해외주식', 'ETF' 버튼을 클릭하여 관심 있는 시장의 종목 리스트를 바로 확인할 수 있습니다.
- **실시간 환율 브리핑**: 상단 바에서 USD(달러), JPY(엔화), EUR(유로) 환율을 실시간으로 확인하세요. 해외 투자 시 가장 중요한 지표입니다.
- **캔들스틱 차트**: 
  - **빨간색**: 주가가 시작가보다 상승했음을 의미합니다.
  - **파란색**: 주가가 시작가보다 하락했음을 의미합니다.
  - 기간(1D, 1W, 1M, 1Y)을 선택해 단기 흐름과 장기 추세를 비교해 보세요.
- **기업 재무 요약**: 시가총액, PER(주가수익비율), ROE(자기자본이익률) 등 기업의 건강 상태를 나타내는 지표를 초보자도 이해하기 쉬운 요약표로 제공합니다.
- **최신 뉴스**: 선택한 종목과 관련된 최신 경제 뉴스를 실시간으로 리스트업해 드립니다.

### 2. 구글 파이낸스 데이터 업데이트 (중요!)
이 웹페이지는 **구글 파이낸스(Google Finance)**의 실제 데이터를 크롤링하여 사용합니다. 가장 최신의 정확한 수치로 업데이트하려면 다음 과정을 따라 하세요.

1. PC에 **Node.js**가 설치되어 있어야 합니다.
2. 터미널(또는 명령 프롬프트)을 열고 프로젝트 폴더로 이동합니다.
3. 다음 명령어를 입력합니다:
   ```bash
   node update_finance_data.js
   ```
4. 수집 및 검증이 완료되면 `market_data.js` 파일이 갱신되며, 웹페이지를 새로고침하면 최신 수치가 반영됩니다.
   - *팁: 데이터 수집 시 허위 내용이나 파싱 오류가 있는지 자동으로 검증 과정을 거칩니다.*

## 🎰 로또픽 (LottoPick) 가이드

번호 고르기 스트레스는 이제 그만! 데이터를 기반으로 나만의 번호를 만들어 보세요.

- **고정수/제외수**: 반드시 포함하고 싶은 번호나, 절대 나오지 않을 것 같은 번호를 필터링할 수 있습니다.
- **홀짝 비율**: 수학적으로 가장 당첨 확률이 높은 홀수와 짝수의 비율(예: 3:3)을 설정할 수 있습니다.
- **보관함**: 마음에 드는 번호 조합은 저장해 두고 메모를 남겨 보세요.
- **자동 당첨 확인**: 저장된 번호가 실제 최신 당첨 번호와 몇 개나 일치하는지 자동으로 계산해 등수를 알려줍니다.

## 🛠 기술 스택
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Visualization**: [ApexCharts](https://apexcharts.com/) (금융 특화 차트 라이브러리)
- **Data Engine**: Node.js Crawler (구글 파이낸스 데이터 추출 및 검증 엔진)
