# DCEO - 주식현황판 및 로또픽

주식현황판과 통계 기반 로또 번호 생성기를 포함한 대시보드 웹 애플리케이션입니다.

---

## 주식현황판 가이드

주식 입문자를 위한 대시보드 사용법입니다.

### 1. 주요 기능
- 자산 카테고리 탭: 국내주식, 해외주식, ETF 카테고리를 전환하며 종목 리스트를 확인할 수 있습니다.
- 환율 정보: 상단 바에서 USD, JPY, EUR 환율 정보를 제공합니다.
- 캔들스틱 차트: 주가의 상승과 하락 흐름을 시각화합니다. 기간 버튼을 통해 일간, 주간, 월간, 연간 추세를 확인할 수 있습니다.
- 기업 재무 요약: 시가총액, PER, ROE 등 주요 재무 지표 요약표를 제공합니다.
- 뉴스 피드: 선택한 종목과 관련된 경제 뉴스를 확인할 수 있습니다.

### 2. 구글 파이낸스 데이터 업데이트
구글 파이낸스의 데이터를 수집하여 웹페이지에 반영하는 방법입니다.

1. PC에 Node.js가 설치된 상태여야 합니다.
2. 터미널에서 프로젝트 폴더로 이동합니다.
3. 다음 명령어를 실행합니다.
   ```bash
   node update_finance_data.js
   ```
4. 수집된 데이터는 market_data.js 파일에 저장되며 웹페이지 새로고침 시 반영됩니다.

## 로또픽 가이드

통계 기반 로또 번호 생성기 사용법입니다.

- 번호 필터링: 포함할 번호와 제외할 번호를 설정하여 생성할 수 있습니다.
- 홀짝 비율: 번호 조합의 홀수와 짝수 비율을 설정할 수 있습니다.
- 보관함: 생성된 번호를 저장하고 메모를 남길 수 있습니다.
- 당첨 확인: 저장된 번호와 최신 당첨 번호를 대조하여 등수를 계산합니다.
